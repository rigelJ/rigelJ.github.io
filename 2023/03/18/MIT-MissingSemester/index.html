MIT-MissingSemester

Description

Name：MIT-MissingSemester

Time：20h

Tips：Bash | Script | Vim | Data-Wrangling | Git | Debuging | Security

Project：https://github.com/rigelJ/Learning-CS-fromScratch/tree/develop/missing_semester

Guides

  • Shell

    • 执行指令

      • echo

        • echo “hello world”

      • $PATH

        • shell通过$PATH的值来寻找echo cd这类执行程序的位置，当我们执行 echo 命令时，shell 了解到需要执行 echo 这个程序，随后它便会在 $PATH 中搜索由 : 所分割的一系列目录，基于名字搜索该程序。当找到该程序时便执行

      • which

        • 判断该程序是在哪个目录被打开的

    • 位置导航

      • 描述计算机里文件的位置

        • 绝对路径
          • 绝对准确的定义一个文件的位置
          • pwd 查看当前路径
        • 相对路径
          • 相对当前路径所在路径
          • .. 上一层目录
          • . 当前目录
          • ~ 用户相关目录
          • 上一个进入的目录，切换目录

      • cd

        • 切换目录
        • 后跟绝对路径或相对路径

      • ls

        • 显示当前目录所有文件
        • -l 文件类型、权限
          • rwx权限
            • 文件
              • r读权限
              • w写权限
              • x执行权限
            • 目录
              • r阅读清单权限
              • w重命名新建删除文件权限
              • x搜索权限

      • mv

        • 重命名或移动文件
        • mv dotfile.md foo.md

      • cp

        • 复制文件
        • cp xxx

      • rm

        • 删除文件
        • rmdir
          • 删除目录

    • 创建程序间的链接

      • 重定向输出流

      • 重定向输入流

      • 追加

      • | 管道符号, 允许我们将一个程序的输出和另外一个程序的输入连接起来：

    • root权限

      • sudo 使用root权限执行指令
      • tee
        • echo 1000 | sudo tee brightness

  • script

    • 变量
    • 变量
      - foo=bar
- echo $foo
      • 定义字符串
        • echo “hello” / echo ‘hello’
        • echo “Value is $foo”
        • 双引号解析，单引号不解析
      • 函数
        • mcd() { mkdir -p “$1” cd “$1”}
        • $开头
          • $0 是脚本名
          • $1是 第一个参数
          • $?是 上一条错误的代码
          • $__是上条命令的最后一个参数
          • $@ 是传给脚本的所有参数的列表
          • $# 是传给脚本的参数个数
          • $* 是以一个单字符串显示所有向脚本传递的参数，与位置变量不同，参数可超过9
          • $$ 是脚本运行的当前进程ID号
          • !! 上一次输入的指令
        • source
      • 输入输出流判断
        • false/true
        • false || echo “Oops”
        • false && echo “This is a code”
        • false ; ok
      • 命令输出存到变量
        • foo=$(pwd)
        • echo “We are in $(pwd)” “”解析
        • cat < (ls) < (ls ..)
      • 条件判断
        • if [[“$i” -ne 0]]; then
      • 缩进展开
        • convert image.{png,jpg}
        • touch foo{1,2,3}
        • touch project{1,2}/src/test/test.{1,2}.py
        • mkdir {foo,bar}/{a..j}
        • diff <(ls foo) <(ls bar)
          • diff -y -W 50 并排格式输出
      • 在shell中使用其他语言
        • use python
          • #!/usr/local/bin/python
          • #!/usr/bin/env python
      • shellcheck
        • 查找bash脚本的错误
      • 帮助类工具
        • man 全
        • tldr 精简
      • 查询类工具
        • 从名称查找
          • find
            • -type 类型
            • -path 路径模式
            • -name 名称
            • -print/print0/printf 输出模式
            • man find
          • fd 精简的find
          • locate 建立数据库而后查找
            • updatedb
        • 从内容查找
          • grep 查找文件
            • grep foobar mcd.sh 在单个文件中查找foobar
            • grep -R foobar . 在指定目录的所有文件中查找foobar
          • fzf 交互式的grep
            • cat script.py | fzf
          • rg
            • rg “import request” -t py ~ 家目录里遍历所有文件，查找有相关表达的文件
            • -C 5 上下文
            • -files-without-match “^#!“
          • ag / ack
        • history 访问shell历史记录
        • 快速导航目录
          • fasd
          • autojump
          • tree
          • broot 交互式的tree
          • nnn
          • ranger
