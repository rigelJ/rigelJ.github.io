Markdown 拓展技巧

HTML元素

不在 Markdown 涵盖范围之内的标签，都可以直接在文档里面用 HTML 撰写。目前支持的HTML元素有：

<kbd>元素：使用Shift +W 执行操作

<b>元素：123

<i>元素：admin

<em>元素：admin

<sup>元素：A1

<br> 元素：换行

转义字符

Markdown 使用了很多特殊符号来表示特定的意义，如果需要显示特定的符号则需要使用转义字，Markdown 使用反斜杠转义特殊字符：

\ * \ * 转义文本加粗
